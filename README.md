# RAG Career Assistant

A Retrieval-Augmented Generation (RAG) based Career Assistant built using Python, Streamlit, FAISS, and LLMs.

## Features
- Career roadmap guidance
- Job skills retrieval
- Document search using RAG
- Resume and job matching (planned)

## Tech Stack
- Python
- Streamlit
- FAISS
- LangChain
- Git & GitHub

## Project Status
Currently under development.
